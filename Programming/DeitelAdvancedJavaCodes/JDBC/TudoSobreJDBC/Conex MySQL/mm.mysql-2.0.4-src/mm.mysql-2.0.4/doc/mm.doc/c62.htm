System Requirements
Installing MM.MySQL

This chapter contains instructions for installing the MM.MySQL drivers on Microsoft Windows and UNIX platforms. If you are on another platform that supports Java and the JDBC API then substitute commands appropriate for your system.

System Requirements

The only requirements for using MM.MySQL are JDK1.1 or compatible and a MySQL server running somewhere that allows access from the host you are going to run MM.MySQL on.
