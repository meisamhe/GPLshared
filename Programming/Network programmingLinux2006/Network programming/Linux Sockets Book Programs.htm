Chapter 17: Sharing Messages with Multicast, Broadcast, and Mbone
bcast_client.c Broadcasting, an older method of sharing information uses the network mask to send and receive general messages. Multicast replaces broadcast as a more WAN-friendly solution. Until multicasting is fully supported on the Internet, Mbone acts as a virtual network using IP-over-IP protocols.
dgram_echo.c
